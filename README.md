# culturasp-adapter
**🎼 CulturaSP-Adapter**: adaptador open source que extrai, estrutura e expõe dados culturais públicos de SP em formatos padroizados. Converte informações estáticas (Sala São Paulo, Pinacoteca, etc.) em experiências dinâmicas: reserva inteligente, cancelamento automático, acessibilidade estruturada e integração multiplataforma
